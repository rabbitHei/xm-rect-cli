## rect-cli  
用于快速创建React项目的脚手架，真正开箱即用直接进行业务开发，目前涵盖下面几种常见业务场景：  
- react + react-router + axios  
- react + react-router + axios(h5版本，vw适配)  
- react + react-router + axios + antd  
- react + react-router + axios + mobx  


## 使用  
```sh
// 全局安装
npm install -g rect-cli

// 新建一个项目
rect myProjectName

// 启动
cd myProjectName
npm start

// 打包
npm run build
```
